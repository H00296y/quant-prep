# quant-prep

量化私募实习Python练习项目。

## 环境
- Python 3.10
- 主要库：pandas, numpy, matplotlib

## 项目列表

### 1. 股票双均线策略回测
- 数据：模拟股票日线数据（随机游走模型）
- 策略：5日均线上穿20日均线买入，下穿卖出
- 输出：累计收益率曲线、交易信号图、夏普比率、最大回撤
- 文件：`01_stock_ma_strategy.py`
- 结果图：`ma_strategy_result.png`

### 2. 可转债低溢价率筛选器
- 数据：模拟可转债数据（50只）
- 筛选：转股溢价率 &lt; 30%，价格 &lt; 130元，到期收益率 &gt; 0%
- 策略：双低值（价格 + 溢价率）排序
- 文件：`02_convertible_bond_screen.py`
- 结果图：`cb_screen_result.png`

- 双均线趋势跟踪策略逻辑
- 可转债双低策略筛选逻辑
