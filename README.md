# quant-prep

量化私募实习前置练习：用最基础的两个策略，走通"数据 → 信号 → 回测 → 绩效评估 → 可视化"这套完整框架。

## 说明

当前版本使用**模拟数据**（随机游走生成的价格 / 随机构造的可转债数据），目的是先把回测框架和工程实现走通、不受真实数据接口限制。下一步计划接入 [akshare](https://github.com/akfamily/akshare) 替换为真实行情数据，详见「后续计划」。

## 项目列表

### 1. 股票双均线策略回测 `01_stock_ma_strategy.py`

- **数据**：模拟股票日线（几何布朗运动生成，约5年，可通过 `seed` 复现）
- **策略**：MA5 上穿 MA20 开仓，下穿平仓——趋势跟踪最基础的形式
- **关键实现细节**：信号用 `shift(1)` 错开一天再计算收益，避免用到未来数据（look-ahead bias）
- **输出**：
  - 控制台：总收益率、买入持有收益、夏普比率、最大回撤、胜率、交易次数
  - 图表：价格+均线+买卖点、累计收益对比、回撤曲线（`ma_strategy_result.png`）

### 2. 可转债双低策略筛选器 `02_convertible_bond_screen.py`

- **数据**：模拟可转债数据（50只，覆盖银行/医药/科技等行业）
- **筛选逻辑**：转股溢价率 < 30% 且 转债价格 < 130元
- **排序**：双低值 = 价格 + 溢价率，取最小的10只
- **输出**：
  - 控制台：全市场概览、筛选结果、双低 Top10
  - 图表：价格-溢价率分布散点图、Top10 柱状图（`cb_screen_result.png`）

## 一个需要说明的局限

双均线策略目前跑在**纯随机游走**生成的数据上。随机游走的收益率理论上不存在自相关，也就是说均线交叉信号在这类数据上本质是噪声交易——现在跑出来的夏普比率、胜率数字是这一组随机种子下的偶然结果，不代表策略在真实市场中有效。这个项目目前的价值是验证**回测框架本身写对了**（信号生成、避免未来函数、绩效指标计算、可视化），策略有效性需要在真实数据上重新检验。

其他已知简化：
- 未考虑交易成本、滑点、冲击成本
- 均线周期（5/20）、筛选阈值（30%/130元）都是经验参数，未做敏感性分析
- 可转债数据未包含到期收益率（YTM），真实计算需要票面利率和付息条款等完整条款数据
