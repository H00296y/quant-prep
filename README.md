# quant-prep

经济学本科大二学生，为量化私募实习准备的 Python 练习项目。

&gt; 注：由于本地网络环境限制（akshare 数据源连接不稳定），本项目部分脚本使用模拟数据演示策略框架与回测逻辑，代码结构可直接替换为真实数据接口。

## 环境

```bash
conda create -n quant python=3.10
conda activate quant
pip install -r requirements.txt
项目列表
1. 股票双均线策略回测
数据：模拟股票日线数据（几何布朗运动随机游走）
策略：5日均线上穿20日均线买入，下穿卖出
说明：本脚本核心目的是演示回测框架（数据读取 → 信号生成 → 收益计算 → 绩效评估 → 可视化）。
反思：纯随机游走序列理论上无自相关性，均线交叉策略在此类数据上不应产生稳定超额收益；实际应用中需先检验标的的动量/趋势特征显著性。
文件：01_stock_ma_strategy.py
2. 可转债双低策略筛选器
数据：模拟可转债数据（50只）
策略：低溢价率 + 低价格（双低值排序）
文件：02_convertible_bond_screen.py
学到的东西
pandas 数据处理与条件筛选
matplotlib 多子图布局
策略回测完整流程：数据 → 信号 → 回测 → 绩效 → 可视化
认知：意识到策略有效性依赖于数据生成过程的前提假设（如趋势存在性），回测前需做序列特征检验
